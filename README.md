# Extensões de Alto Desempenho em PHP

Este repositório faz parte do material de apoio do livro **"Extensões PHP de Alto Desempenho - Rust + PHP: Performance Nativa Sem Complexidade de C/C++"**, ISBN (Livro Digital): **978-65-01-89387-7**.

O principal objetivo do livro é popularizar o desenvolvimento de extensções para PHP usando Rust, e demonstrar o real ganho de desempenho, comprarando diretamente:

- PHP Puro
- PHP + Rust (single thread)
- PHP + Rust (multi thread)

E o objetivo deste repositório é fornecer um exemplo totalmente funcional dos códigos comentados no livro, permitindo ao desenvolvedor executar de forma muito mais fácil, evitando erros de digitação ao copiar os códigos do livro.

# Conhecimento

O livro não é essencial para compreender esse repositório, entretando é altamente recomendada a leitura para se familiarizar com os conceitos aqui utilizados, e entender os motivos dos códigos serem escritos desta forma. Especialmente se você é programador PHP e não tem nenhuma familiaridade com Rust.
